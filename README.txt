Sentiment Analysis with Naive Bayes
Overview
This project aims to classify movie reviews as either positive or negative using a Naive Bayes classifier. The reviews are sourced from the Cornell Movie-Review Dataset. It’s a straightforward sentiment analysis task that uses natural language processing techniques to understand the emotions behind the reviews.

Data
The dataset is already included in the data folder, so you don’t need to download anything extra.
Requirements

To get started, you’ll need to install the necessary packages. Just run:
pip3 install -r requirements.txt  (This will make sure all the dependencies are in place.)


Running the Project
To run the project, follow these steps:

Step 1:-

Navigate to the source code directory:
cd src

Step 2:-

Run the Naive Bayes sentiment analysis script:
python3 naive_bayes.py
